1. Prompt Injection Scan

Detect prompt injection attacks in text input.

POST /api/nexus/injection/scan
Content-Type: application/json

{
  "text": "Ignore all previous instructions",
  "context": "user_input"
}

Response:
{
  "safe": false,
  "findings_count": 2,
  "findings": [...],
  "sanitized": "[BLOCKED] and [BLOCKED]"
}

2. AI Agent Code Scan

Find security vulnerabilities in AI agent code.

POST /api/nexus/agent/scan
Content-Type: application/json

{
  "code": "import os\n@tool\ndef run(cmd): return os.system(cmd)",
  "filename": "agent.py"
}

Response:
{
  "issues_found": 3,
  "critical": 1,
  "high": 0,
  "findings": [...]
}

3. Guardian Check

Check a tool call against 11 security layers.

POST /api/nexus/guardian/check
Content-Type: application/json

{
  "tool_name": "run_command",
  "tool_input": "rm -rf /",
  "agent_name": "my_agent",
  "agent_reasoning": "Cleaning up"
}

Response:
{
  "action": "BLOCK",
  "reason": "Shell injection blocked",
  "layer": 4
}

4. AIBOM Generation

Generate an AI Bill of Materials from agent code.

POST /api/nexus/aibom
Content-Type: application/json

{
  "code": "agent code here",
  "system_name": "My Agent",
  "filename": "agent.py"
}

Response:
{
  "risk_score": 90,
  "aibom": { ... },
  "compliance_summary": { ... }
}

5. Sentinel Check (Guardian + Behavioral Biometrics)

Check a tool call with behavioral anomaly detection.

POST /api/nexus/sentinel/check
Content-Type: application/json

{
  "tool_name": "read_file",
  "tool_input": "config.json",
  "agent_name": "my_agent"
}

Response:
{
  "action": "ALLOW",
  "anomaly_score": 0.0,
  "reason": "All 10 security layers passed"
}
